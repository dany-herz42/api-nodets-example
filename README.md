This is a very basic example that I used to integrate TypeScript with a Node.js and Express api.

<br/>
<br/>

### Try it
1. Clone this repository
2. Open a terminal and install dependencies:
```
> npm install
```
3. Run development server:
```
> npm run dev
```
4. Or run the server
```
> npm start
```
